# 🎓 KUSE Activity & News Web Application

Web Application สำหรับแสดงข้อมูลข่าวสารและกิจกรรมของมหาวิทยาลัย  
พัฒนาเป็นส่วนหนึ่งของ **Senior Project**

---

## 🚀 Project Overview
โปรเจกต์นี้เป็นผลงานในช่วงชั้นปีที่ 3  
ที่มหาวิทยาลัยเกษตรศาสตร์ เฉลิมพระเกียรติ สกลนคร (KUSE)

ออกแบบและพัฒนา Web Application  
เพื่อเป็น **ศูนย์กลางรวมข้อมูลข่าวสารและกิจกรรม**

---

## 🎯 Features
- ระบบค้นหาข้อมูล (Search)
- ระบบกรองข้อมูล (Filter)
- แสดงรายการกิจกรรม (Activity List)
- หน้ารายละเอียดกิจกรรม (Detail Page)
- Responsive Design
